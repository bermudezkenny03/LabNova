# Lab Nova

Este repositorio contiene la aplicación Lab Nova, un sistema de reservas para equipos y laboratorios.

## Documentación técnica

Toda la documentación técnica se encuentra en la carpeta `docs/`.

- `docs/README.md` — Entrada principal de la documentación.
- `docs/architecture.md` — Arquitectura y diseño.
- `docs/setup.md` — Configuración e instalación local.
- `docs/api.md` — Endpoints de la API.
- `docs/deployment.md` — Flujo de despliegue y Vercel.
- `docs/testing.md` — Pruebas y comandos de validación.
- `docs/quality-guidelines.md` — Buenas prácticas de calidad.

## Carpetas principales

- `frontend-lab-nova/` — Aplicación React.
- `backend-lab-nova/` — API Laravel.
